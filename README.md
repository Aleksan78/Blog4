# Blog Engine API

Учебный проект полнофункционального API для системы управления блогом.

##  Технологический стек
* **Платформа:** .NET 8
* **Архитектура:** Многослойная (DAL, BLL, Web API)
* **База данных:** Entity Framework Core (MS SQL Server / SQLite)
* **Документация:** Swagger / OpenAPI с XML-комментариями
* **DI:** Внедрение зависимостей через первичные конструкторы (Primary Constructors)

##  Реализованный функционал
В рамках данного этапа были разработаны следующие контроллеры:
* **Account** — регистрация, авторизация и управление пользователями.
* **Posts** — создание, чтение, обновление и удаление статей.
* **Tags** — управление тегами для классификации контента.
* **Comments** — добавление и модерирование комментариев.
* **Roles** — управление ролями доступа (Администратор, Модератор и т.д.).

##  Запуск проекта
1. Склонируйте репозиторий.
2. Убедитесь, что строка подключения в `appsettings.json` проекта API верна.
3. Выполните команду `dotnet build`.
4. Запустите проект API.
5. Перейдите по адресу `/swagger`, чтобы открыть интерактивную документацию.

##  Особенности реализации
* Реализована полная интеграция с существующим слоем бизнес-логики (BLL).
* Настроена автоматическая генерация документации API на основе XML-комментариев в коде.
* Код очищен от шаблонных файлов (WeatherForecast) и технических предупреждений компилятора.
